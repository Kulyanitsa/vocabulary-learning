# 📚 Vocabulary Learning

Веб-приложение для изучения английских слов с карточками, викториной и режимом контекста.

**Живая версия:** https://kulyanitsa.github.io/vocabulary-learning/

## Возможности

- **Карточки** — переворачиваемые флэш-карточки с произношением (IPA) и авто-озвучкой
- **Викторина** — выбор правильного перевода из 4 вариантов
- **Контекст** — вспомни слово по фразе (вставь пропущенное слово)
- **Статистика** — прогресс по каждому слову, экспорт в JSON
- **Тёмная тема** — переключается кнопкой, запоминается

## Локальный запуск

Приложение использует `fetch()` для загрузки словаря, поэтому нужен локальный сервер:

```bash
# Python 3
python3 -m http.server 8080

# Node.js (npx)
npx serve .
```

Затем открой [http://localhost:8080](http://localhost:8080).

## Добавление слов

Отредактируй [vocabulary.json](vocabulary.json) — добавь новый объект:

```json
"word": {
  "ipa": "/wɜːrd/",
  "translation": "перевод",
  "context": "фраза с word"
}
```

После коммита в `main` GitHub Actions автоматически обновит сайт.
